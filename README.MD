# 毕业设计
    本项目以赛讯为主题,提供给用户查询信息,以及交流信息的平台。使用Intellij Idea开发,编程语言为Java,服务端使用J2EE+Servlet+JavaBean交互,并且通过JDBC操作数据库,数据库采用的是MYSQL
